# Prompt Engineering Workflow Project

## Project Overview

This project demonstrates a complete Prompt Engineering Workflow using Google Gemini AI API and Python.

The main goal of this project is to improve AI-generated responses by creating better prompts, testing outputs, evaluating response quality, and refining prompts step by step.

The project focuses on explaining technical concepts in a simple and beginner-friendly way.

---

# Project Objective

The objective of this project is to:

- Create multiple prompt versions
- Generate AI responses automatically
- Evaluate the quality of outputs
- Compare results between prompts
- Improve prompts based on evaluation results

This project shows how prompt quality directly affects AI-generated responses.

---

# Selected Topic

### Recommendation Systems

The project uses the topic "Recommendation Systems" to test how AI explanations can be improved for beginner-level users.
